# Agents

HANTER comes with five built-in agents.

## DevAgent
Full-stack development agent for code generation, debugging, and deployment.

**Capabilities:**
- Code generation (React, Python, etc.)
- Project scaffolding
- Git operations
- Deployment to Vercel/Netlify
- Debugging and testing

**RAM Budget:** 100MB

## SocialAgent
Social media management agent.

**Capabilities:**
- Content creation
- Post scheduling
- Engagement analytics
- Hashtag generation

**RAM Budget:** 80MB

## WebAgent
Web automation agent.

**Capabilities:**
- Web scraping
- URL monitoring
- Web research
- Content archiving

**RAM Budget:** 60MB

## FileAgent
File management agent.

**Capabilities:**
- File organization
- File search
- Format conversion
- Sync and backup

**RAM Budget:** 40MB

## SysAgent
System automation agent.

**Capabilities:**
- System monitoring
- Task automation
- Cleanup and optimization

**RAM Budget:** 50MB
