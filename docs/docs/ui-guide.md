# UI Guide

## Dashboard Layout

The HANTER interface features a cyberpunk-themed dashboard:

- **Sidebar**: Navigation and agent status
- **TopBar**: System metrics and connection status
- **Main Area**: Context-dependent views
- **Command Center**: Text/voice input and live log stream

## Theme

```css
--bg-primary: #0a0a0f
--bg-secondary: #12121a
--accent-cyan: #00f0ff
--accent-pink: #ff006e
--accent-green: #00ff88
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send command |
| `Ctrl+K` | Focus command input |
| `Ctrl+B` | Toggle sidebar |
