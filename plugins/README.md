# HANTER Plugins

This directory contains community plugins and skills for HANTER.

## Structure

Each plugin should be in its own subdirectory with a `manifest.yaml`:

```yaml
name: my-plugin
version: 1.0.0
description: My awesome plugin
author: Your Name
entry: plugin.py
```

## Availability

Plugins not yet available. The plugin system is coming in v1.1.
