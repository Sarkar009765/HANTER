# Changelog

## [1.0.0] - 2026-05-25

### Added
- Initial release of HANTER
- Tauri v2 desktop app with React 19 + cyberpunk UI
- Python FastAPI backend with WebSocket real-time communication
- 5 specialized agents: DevAgent, SocialAgent, WebAgent, FileAgent, SysAgent
- Skill system with lazy loading and auto-unload (300s idle timeout)
- 2GB RAM optimization protocol with emergency mode
- SQLite short-term memory + ChromaDB long-term vector memory
- LiteLLM integration (Groq, Together, Ollama)
- Secure sandbox for command/code/file execution
- Voice command support (Web Speech API)
- 3D neural network visualizer (Three.js)
- Cross-platform build support (Windows, macOS, Linux)
